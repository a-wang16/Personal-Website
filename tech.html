An earlier project with many errors, this was one of the first steps into the coding scene and sparked a passion to combine fun, interactivity, programming, and utility.   • (WIP) HTTP Web Crawler/ScuffedCrawler: A C++ project for my Networks and Distributed Processing class that utilizes HTTP1.0 calls. A marriage between a hobby of mine and programming, this project helps create a bridge to fine-tune virtual avatar software into accepting or enabling a larger range of expressions.
